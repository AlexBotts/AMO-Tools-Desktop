Help

Choose a log file from your file system.
Select the format of the date the file is using.
Once you have submitted your log file, a table with the first row of data will appear.
Click "Add Additional .csv Data" to import data from addtional .csv files
If no date field exists, you can click "Continue Without Day Type Analysis" and the "Day Type Analysis" portion of the application will be disabled but you can still use the "Data Visualization" portion.
"Use Column" selects which data columns will be inlcuded in the "Day Type Analysis" and "Data Visualization" sections.
"Column Name" is the name corresponding to the column header as it appears in the imported .csv file.
An "Alias" can be set that will be used for display purposed in the "Day Type Analysis" and "Data Visualization" sections.
"Units" can be set that correspond to the column.
Once you have cleaned your data, click "Submit" and the application will do some number crunching. Once this is complete you can continue to "Day Type Analysis" or "Data Visualization".
