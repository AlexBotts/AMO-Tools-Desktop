Help

Graph Basics
Set chart title and plot type.
For Histogram plot's select if data should use a standard deviation or a specific number of bins.
To create a time series use a scatter plot and select a date field for the x-axis.
X Axis
Set x-axis label
Add <br> to create a new line
Wrap your text in <b></b> to bold it.
Wrap your text in <i></i> to italisize it.
Choose the data you wish to use for the x-axis.
Y-Axis
Set y-axis label
Add <br> to create a new line
Wrap your text in <b></b> to bold it.
Wrap your text in <i></i> to italisize it.
Use the text box to set a name for the corresponding data
Choose the data you wish to use for the x-axis.
Click "+Add Another Data Series" to add additional y-axis data
When additional data is added, a second y-axis with a different scale can be used on the right hand side of the plot
Click the series color box to choose what color you want the corresponding data to appear on the plot
Annotations
Click the chart to select the data you want to annotate
Fill the text box with the desired annotation
Add <br> to create a new line
Wrap your text in <b></b> to bold it.
Wrap your text in <i></i> to italisize it.
You can customize the font size, font color, background color, arrow color and arrow size
Use the arrow buttons to move the location of the annotation
