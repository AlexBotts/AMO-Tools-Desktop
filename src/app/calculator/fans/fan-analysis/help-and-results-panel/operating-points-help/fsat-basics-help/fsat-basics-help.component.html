Basic Fan Info Help
This calculator lets the user determine the power, flow rate, pressure and fan efficiency at set operating points. It also helps the user perform a pitot tube analysis for the determination of flow rate and pressure.
This section sets up the basic information about the fan.
Fan Speed
The operating or nameplate speed of the fan.
Motor Speed
The nameplate speed of the motor.
Barometric Pressure
The pressure exerted by the weight of air in the atmosphere. Can be estimated with information on the fan elevation above sea level.
Corrected Fan Speed
Fan speed that calculated values (power, flow, pressure) will be corrected to. Usually the rated fan speed or fan speed used for manufacturer’s fan curve.
Corrected Density
Gas density that calculated values (power, flow, pressure) will be corrected to. Usually the gas density used for manufacturer’s fan curve.
Corrected Barometric Pressure
Barometric Pressure that calculated values (power, flow, pressure) will be corrected to. Usually the barometric pressure used for manufacturer’s fan curve.
Rated Fan Performance Includes Evase
Specify if a discharge evase has been accounted for in the rated fan performance by selecting ‘Yes’ or ‘No.’
Number of Traverse Planes
1 is the default setting, but if more than one traverse plane is required to capture the total flow of the fan, then select 2 or 3 (more than 3 is rare and thus not included).
