Plane Information
This section is where the user more specific data about the fan system (e.g., ductwork arrangement and dimensions, the fan).
This requires the user to enter data about the overall system (any system effects to take into consideration) and any pressure drops between the fan inlet/outlet and the points of measurement. If the user can measure the static pressure of the fan inlet or outlet directly, this pressure loss is zero and values for static pressure should be entered on planes 4 or 5.
Plane 1
The first plane is the inlet of the fan.
Plane 2
NEEDED
Plane 3
Plane 3 is the plane used for the Pitot-static traverse for purpose of determining flow rate.
Plane 4
Plane 4 is a plane upstream of fan used to establish fan inlet Plane Static Pressure
Plane 5
Plane 5 is a plane downstream of fan used to establish fan outlet Plane 2 Static Pressure
Total Pressure Loss Between Planes 1 and 4
Input a positive value for the pressure loss between planes 1 and 4. This loss may be established from the frictional/dynamic losses as per ASHRAE Fundamentals ‘Duct Design’, or by some other means.
