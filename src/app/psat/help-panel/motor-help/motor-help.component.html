Motor Help

Enter measured data to calculate your pump and system's optimization rating and annual savings potential.


Watch tutorial on gathering motor data.

Line Frequency

Line Frequency is the mains supply frequency.

Options: 50 Hz or 60 Hz

Horse Power

Horse Power represents the rated power for the motor.

Units: HP
Minimum Value Allowed: 0
Maximum Value Allowed: None

Motor RPM

Motor RPM is the speed of the motor.

Units: RPM
Minimum Value Allowed: 0
Maximum Value Allowed: None

Efficiency Class

Efficiency Class is the classification of efficiency given to motor. This can be found either on the nameplate or based on how the motor compares with the NEMA MG 1-2003.

Options:

  • Standard Efficency
  • Energy Efficient
  • Average
  • Specified (below)
Voltage

Measured bus voltage.

Units: Voltage
Minimum Value Allowed: 0
Maximum Value Allowed: None

Full Load Amps

Full Load Amps represent either the known or estimated amps used as a normalizing value.

Units: amps
Minimum Value Allowed: 0
Maximum Value Allowed: None

Size Margin

This Size Margin is added to the motor selected for the optimal application.

Units: Percent
Minimum Value Allowed: 0
Maximum Value Allowed: 100
