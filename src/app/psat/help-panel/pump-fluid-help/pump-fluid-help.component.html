Pump Type

Pump Type represents what style of pump is being used based off of the listings in the standard ANSI/HI 1.3-2000.

This value will be used to estimate achievable pump efficiencies based on pump style and operating conditions.


  • End Suction Slurry
  • End Suction Sewage
  • End Suction Stock
  • End Suction Submersible Sewage
  • API double Suction
  • Multistage Boiler Feed
  • End Suction ANSI/API
  • Axial Flow
  • Double Suction
  • Vertical Turbine
  • Large End Suction
  • Specified Optimal eff (below)
Specified Pump Efficiency

The specified pump efficiency input is only displayed when the pump style is "Selected optimal eff (below)".

The input value would normally be the pump efficiency at the specified operating conditions (under the Field data section), as opposed to the best efficiency point flow rate.

Units: Percent (%)
Minimum Value Allowed: 0
Maximum Value Allowed: 100

Pump Speed

The operating or nameplate speed at which the pump is running at. Used along with the measure/required flow rate, head, and number of stages, to calculate the pump specific speed. The specific speed is used to determine efficiency penalty associated with the particular pump application.

Units: RPM
Minimum Value Allowed (Direct Drive): 540
Maximum Value Allowed (Direct Drive): 3960
Minimum Value Allowed (Dir Drive): 1
Maximum Value Allowed (Belt Drive): —

Drive

This drop-down selection menu allows the user to define whether the pump is direct driven by the motor or belt-driven.

This value is used in the Load Estimation calculation. Note: The current estimating method should NOT be used if adjustable frequency drives are used.

Options: Direct Drive or Belt Drive

Specific Gravity

Specific Gravity is the ratio of the density of the fluid to water at standard condition.

This is used in calculating the fluid power at the specified pump flow rate and head conditions. This value can be found through fluid property tables.

Minimum Value Allowed: 0
Maximum Value Allowed: None

Stages

The number of pump stages. This value is used to calculate pump specific speed.

Minimum Value Allowed: 0
Maximum Value Allowed: None
