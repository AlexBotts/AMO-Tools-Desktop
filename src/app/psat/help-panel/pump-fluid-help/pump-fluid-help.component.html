Pump Type

Pump Type represents what style of pump is being used based off of the listings in the standard ANSI/HI 1.3-2000.

Options:

  • End Suction Slurry
  • End Suction Sewage
  • End Suction Stock
  • API double Suction
  • Multistage Boiler Feed
  • End Suction ANSI/API
  • Axial Flow
  • Double Suction
  • Vertical Turbine
  • Large End Suction
  • Specified Optimal eff (below)
Pump Speed

The operating or nameplate speed at which the pump is running at.

Units: RPM
Minimum Value Allowed: 0
Maximum Value Allowed: None

Drive

This drop-down selection menu allows the user to define whether the pump is direct driven by the motor or belt-driven.

Options: Direct Drive or Belt Drive

Kinematic Viscosity

Kinematic Viscosity is the viscosity of the fluid being pumped in centistokes.

Units: cS
Minimum Value Allowed: 0
Maximum Value Allowed: None

Specific Gravity

Specific Gravity is the ratio of the density of the fluid to water at standard condition.

Minimum Value Allowed: 0
Maximum Value Allowed: None

Stages

The number of pump stages.

Minimum Value Allowed: 0
Maximum Value Allowed: None

Fixed Specific Speed

The Fixed specific speed switch allows the user to specify whether a specific speed penalty that is a part of the HI achievable efficiency algorithm sequence, is actually imposed.

Options: Yes or No
