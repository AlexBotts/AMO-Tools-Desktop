Field Data Help

Enter measured feild data to calculate your system's optimization rating and annual savings potential.

Watch tutorial on gathering field data.

Operating Fraction

Operating Fraction represents the fraction of calender hours the equipment is running.

Units: Percent
Minimum Value Allowed: 0
Maximum Value Allowed: 100

Cost (KwHr)

Cost is the per unit cost of electricity in (kw/hr).

Units: Money
Minimum Value Allowed: 0
Maximum Value Allowed: None

Flow Rate (gpm)

Flow Rate represents either the measured or required rate of flow.

Units: GPM, MGD, m^3/h, L/s
Minimum Value Allowed: 0
Maximum Value Allowed: None

Head

Head represents either the measured or required pump head in feet or meters.

Units: Feet or Meters
Minimum Value Allowed: 0
Maximum Value Allowed: None

Load Estimated Method

Load Estimation Method can either be one of two choices: Power or Current. Both of which calculate Load.

Options: Power or Current

Motor (kW)

Motor represents either a measured or average current among the three phases.

Units: kW or amps
Minimum Value Allowed: 0
Maximum Value Allowed: None

Voltage

Measured bus voltage.

Units: Voltage
Minimum Value Allowed: 0
Maximum Value Allowed: None
