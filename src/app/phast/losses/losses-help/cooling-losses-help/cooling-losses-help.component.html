Cooling Medium

Select water or air from the drop down list. The data requirement will change accordingly.

Inlet Temperature (℉)

Inlet temperature of water/air as it enters water/air cooled parts. Use temperature recorder/indicator reading or actual measurement using a temperature gage, thermocouple or estimate based on measurements of water temperature at any location in the plant. This is an average value for the selected section if multiple water/air cooled sections are considered.

Outlet Temperature (℉)

Outlet temperature (Celsius) of water/air as it leaves water/air cooled parts. Use temperature recorder/indicator reading or actual measurement or estimate based on measurements of water temperature at any location downstream before water is mixed with other flows. This is an average value for the selected section if multiple water/air cooled sections are considered.
