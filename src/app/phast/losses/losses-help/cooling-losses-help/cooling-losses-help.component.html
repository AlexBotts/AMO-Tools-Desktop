Cooling Medium

Select the appropriate cooling medium: gas or liquid. The default values for gas represent air, the default values for liquid represent water.

Inlet temperature

The temperature of the charge material as it enters the unit. Use production data or actual measurement.

Units: ℉ or ℃
Minimum Value Allowed: 0

Inlet temperature

The temperature of the charge material as it enters the unit. Use production data or actual measurement.

Units: ℉ or ℃
Minimum Value Allowed: 0

Correction Factor

Engineering correction factor multiplied to the result to account for a known systematic error.

Minimum Value Allowed: 0

Gas Flow

Volumetric flow rate of gas per hour. Note that for batch furnaces the total weight should be corrected for cycle time to obtain weight per hour value.

Units: SCFM or Nm3/min
Minimum Value Allowed: 0

Gas Density

Density of the cooling gas. Default value is for air; may be more accurate to compute as the average of density based on inlet and outlet temperature.

Units: lb/SCF or kg/Nm3
Minimum Value Allowed: 0

Specific Heat

Amount of heat per unit mass required to raise the temperature by one degree of temperature. Default value is for air; may be more accurate to compute as the average of specific heat based on inlet and outlet temperature.

Units: Btu/(lb-℉) or kJ/(kg-℃)
Minimum Value Allowed: 0

Liquid Flow

Volumetric flow rate of liquid per hour. Note that for batch furnaces the total weight should be corrected for cycle time to obtain weight per hour value.

Units: gal/min or L/min
Minimum Value Allowed: 0

Density

Density of the cooling liquid. Default value is for water; may be more accurate to compute as the average of density based on inlet and outlet temperature

Units: lb/gal or kg/L
Minimum Value Allowed: 0

Average Specific Heat

Amount of heat per unit mass required to raise the temperature by one degree of temperature. Default value is for water; may be more accurate to compute as the average of specific heat based on inlet and outlet temperature.

Units: Btu/(lb-℉) or kJ/(kg-℃)
Minimum Value Allowed: 0
