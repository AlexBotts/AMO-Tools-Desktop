Number of Openings of Same Size and Shape

Give number of openings of the same shape and size.

Furnace Wall Thickness (Inch)

Value of wall thickness (insulation - refractory etc.) at the location of openings in the wall or doors etc. Use actual measurements if possible or information from the current furnace drawings.

Length of Openings (Inch) Or Diameter of Openings (Inch)

Value of length of the openings for square or rectangular openings in the wall or doors etc. Use actual measurements if possible or information from the current furnace drawings. Use value for a typical opening or average value. This value is used only to calculate view factor and not for calculation of heat loss area.

Height of Openings (Inch)

Value of height of the openings for square or rectangular openings in the wall or doors etc. Use actual measurements if possible or information from the current furnace drawings. Use value for a typical opening or average value. This value is used only to calculate view factor and not for calculation of heat loss area.

View Factor - User defined if known or from chart
Avg. Zone or Radiation Source Temp. (℉)

Use the furnace zone temperature at the location of the openings. This can be obtained from furnace temperature recorder-indicator. Use an 'average" value of temperature if the openings are in different zones or at different locations.

Ambient Temp. (℉)

Temperature of the area outside the furnace where the openings are located. In most cases this is the ambient temperature in the area where the furnace and openings are located.

Emissivity of the Source (Typical - 0.9)

Emissivity of the radiation source (usually interior of the furnace). The default or suggested value is 0.9

% of Time Open

This represents the time during which the furnace interior is exposed or open to the ambient and heat is lost by radiation. it is expressed as percentage of the total operating time or cycle time. Typical examples including opening of a door for charging and discharging load or opening of a damper for part of the operating time.
